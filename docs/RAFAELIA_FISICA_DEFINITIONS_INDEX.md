# RAFAELIA FISICA DEFINITIONS INDEX

> Índice inicial para transformar o repositório `Fisica` em base de definições, unidades, limites, variáveis e fundamentos verificáveis.

## Função do repositório

```text
Fisica = base de definições técnicas para sustentar Cosmos, RLL e Catalogo-cosmologico.
```

Este repositório deve priorizar clareza conceitual, unidades, domínios de validade e separação entre física conhecida, hipótese autoral e metáfora.

---

## Regimes principais

| Regime | Descrição | Uso correto |
|---|---|---|
| `[FÍSICA-CONHECIDA]` | equação ou conceito estabelecido | citar fonte primária ou livro-texto |
| `[DEFINIÇÃO]` | termo, variável, unidade, convenção | estabilizar vocabulário |
| `[HIPÓTESE-AUTORAL]` | proposta RafaelIA/RLL ainda não validada | apontar teste e limite |
| `[MODELO-EFETIVO]` | aproximação útil dentro de domínio limitado | declarar domínio e erro |
| `[SIMBÓLICO]` | linguagem espiritual/metafórica | não substituir definição técnica |
| `[LIMITE]` | restrição, incerteza, não validação | proteger contra claim indevido |

## Critério para aceitar fórmula

Toda fórmula deve trazer:

```text
1. nome;
2. expressão;
3. variáveis;
4. unidades;
5. domínio de validade;
6. fonte ou status autoral;
7. exemplo mínimo;
8. limite de interpretação.
```

## Regra central

```text
Sem unidade, é símbolo ou hipótese.
Com unidade e domínio, pode virar modelo.
Com dado e métrica, pode virar validação.
```
