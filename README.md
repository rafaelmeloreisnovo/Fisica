## Regra de evidência

```text
conceito != modelo validado != simulação reproduzível
         != protótipo != ensaio independente != patente concedida
```

Estados aceitos: `REFERENCE`, `HYPOTHESIS`, `SIMULATED`, `MEASURED`, `REPLICATED`, `IP_CANDIDATE` e `TOKEN_VAZIO`.
